Instructions for Linux
1) Both linux folder files are executables for Linux
2) Execute them by using terminal 
3) unzip wifiaudio.zip
4) Open terminal (Konsloe / GNOME terminal) in Graphical mode
5) cd <path where binaries are extracted>
6) cd linux
7) ./wifiaudio_linux_32bit (For 32 bit Linux) or ./wifiaudio_linux_64bit (For 64bit Linux)
8) Enter IP address of Mobile device and then press start
